ЛАЙФХАК: Настройка Stability в ElevenLabs Eleven v3

ИНСТРУМЕНТ: ElevenLabs (модель Eleven v3)

ПРОБЛЕМА:
Голос в TTS либо звучит как робот без эмоций, либо начинает "галлюцинировать" — менять интонацию, вставлять странные звуки, иногда обрывается. Большинство просто перегенерирует пока не повезёт.

РЕШЕНИЕ — три режима Stability в v3:

CREATIVE
- Максимум эмоций, выразительности
- Но может галлюцинировать — добавлять то, чего нет в тексте
- Используй когда: персонажная озвучка, эмоциональные сцены, рекламные ролики

NATURAL
- Баланс между эмоциями и стабильностью
- Ближе всего к оригинальному голосу
- Используй когда: подкаст, нарратив, аудиокниги — большинство задач

ROBUST
- Максимальная стабильность
- Но менее отзывчив на audio-теги типа [whispering] или [excited]
- Используй когда: длинные дикторские тексты, документалки, ТТС-агенты

ВАЖНЫЕ МОМЕНТЫ:
- Style Exaggeration держи 3-5% — большие значения ломают голос
- В v3 нет SSML break tags — используй троеточие (...) для пауз и тире (—) для коротких разделений
- v3 хуже работает с Professional Voice Clones — для v3 лучше Instant Voice Clones

КЕЙСЫ ИЗ ПРАКТИКИ:
- Озвучка YouTube-видео — Natural
- Аудиокнига с диалогами персонажей — Creative для эмоций
- AI-агент для бизнеса — Robust для предсказуемости
